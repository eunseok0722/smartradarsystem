Software Defined Imaging Radar

Leading Software Defined Imaging Radar

  • SDIR refers to the state that the value of software in a vehicle which exceeds that of the mechanical hardware: a radar sensor unit.

  • This reflecst the digital transformation of vehicles from rigid(fixed performance) machines to expandable mobile electronics that can be continuously upgraded by OTA: over the air.

  • To become such intelligent sensor, the structure is pre-embedded with advanced hardware before standard operating procedures (SOP)

  • SRS supports the SDIR structure which is pre-embedded with advanced hardware that OEMs/Tier-1s can apply their fresh algorithm via OTA.
